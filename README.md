## 🧠 O Que Estou Aprendendo

A trilha de aprendizado do curso foi estruturada para construir o raciocínio lógico do básico ao avançado. Entre os principais conceitos absorvidos e aplicados estão:

* **Fundamentos de Lógica:** Manipulação de variáveis, tipos de dados (caractere, numérico, lógico) e conversões de tipos[cite: 1].
* **Estruturas de Decisão Condicional:** Aplicação prática de condicionais simples, compostas e encadeadas (`IF / ELSEIF / ELSE`) para controle de fluxo[cite: 1].
* **Seleção Múltipla:** Otimização de menus e checagem de variáveis utilizando a estrutura `DO CASE / OTHERWISE`[cite: 1].
* **Modularização:** Organização de código limpo dividindo o sistema em procedimentos (`PROCEDURE`) e funções (`FUNCTION`) com passagem de parâmetros[cite: 1].
* **Operadores Avançados:** Uso de operadores relacionais, lógicos (`.AND.`, `.OR.`, `.NOT.`) e funções nativas de manipulação de texto e matemática[cite: 1].
