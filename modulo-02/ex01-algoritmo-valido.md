# Exercício 1 — Isto é um algoritmo válido?

1. **Some dois números**

Não é válido, pois está ambíguo e incompleto. Não informa quais são os números e não informa o que fazer com o resultado.

2. **Leia um número. Se for positivo, mostre ‘positivo’. Senão, mostre ‘negativo’.**

É válido, o passo a passo está definido, deixando claro a tomada de decisão.

3. **Fique repetindo até acabar.**

Não é válido, é ambíguo, pois nós dá apenas uma instrução de repetição o que pode leva a um loop infinito, pois não tem um instrução de parada.

4. **Pegue um ingrediente qualquer e cozinhe.**

Não é válido, a instrução é ambígua, para o computador as instruções precisam ser exatas, ele não irá entender apenas a instrução pegar um ingrediente qualquer.
