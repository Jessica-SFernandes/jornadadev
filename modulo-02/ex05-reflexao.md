# Exercício 5 — Reflexão

1. Por que dizemos que **“o computador não erra”?** O que isso implica para quem programa?

Não erra, porque ele vai executar exatamente as instruções dadas a ele de maneira literal, mesmo se houver algum erro, para quem programa isso traz uma grande responsabilidade, pois a todo o planejamento e os testes vai partir de quem vai dar o comando para o computador.
