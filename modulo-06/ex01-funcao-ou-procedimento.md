# Exercício 1: Função ou Procedimento?

1. **`Str()`** -> **Função**
        Converte e devolve um valor numérico formatado como string de texto.
2. **`QOut()`** -> **Procedimento**
        Apenas exibe dados na tela e pula para a próxima linha.
3. **`Date()`** -> **Função**
        Retorna o valor do tipo data.
4. **`Len()`** -> **Função**
        Retorna um número inteiro.
5. **`Upper()`** -> **Função**
        Retorna uma nova string com todos os caracteres alfabéticos convertidos em maiúsculas.
6. **`ClearScreen()`** -> **Procedimento**
        Apenas executa a ação de limpar o terminal.
