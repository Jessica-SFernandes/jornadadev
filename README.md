# Jornada DEV START — TOTVS 🚀

Olá! Este repositório foi criado para registrar toda a minha evolução, códigos e projetos práticos desenvolvidos durante a **Jornada DEV START**, um programa de formação promovido pela TOTVS Paulista. 

O objetivo principal desta jornada é construir uma base sólida em lógica de programação e desenvolvimento de software, utilizando a linguagem **Harbour** como ferramenta fundamental de aprendizado.

## 🧠 O Que Estou Aprendendo

A trilha de aprendizado do curso foi estruturada para construir o raciocínio lógico do básico ao avançado. Entre os principais conceitos absorvidos e aplicados estão:

* **Fundamentos de Lógica:** Manipulação de variáveis, tipos de dados (caractere, numérico, lógico) e conversões de tipos.
* **Estruturas de Decisão Condicional:** Aplicação prática de condicionais simples, compostas e encadeadas (`IF / ELSEIF / ELSE`) para controle de fluxo.
* **Seleção Múltipla:** Otimização de menus e checagem de variáveis utilizando a estrutura `DO CASE / OTHERWISE`.
* **Modularização:** Organização de código limpo dividindo o sistema em procedimentos (`PROCEDURE`) e funções (`FUNCTION`) com passagem de parâmetros.
* **Operadores Avançados:** Uso de operadores relacionais, lógicos (`.AND.`, `.OR.`, `.NOT.`) e funções nativas de manipulação de texto e matemática.
